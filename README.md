# python-audio-tools
A collection of audio tools in python.

## Usage
Please refer to [/code/examples.py](examples.py).

## Functions
### Display
* Waveform
* Spectrum

### Transcoder
* FFmpeg control tools (wav, mp3, opus, vorbis-ogg, volume detection)
* Lame control tools (mp3)
* Opus control tools (opus)
